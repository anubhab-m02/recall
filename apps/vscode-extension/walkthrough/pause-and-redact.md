**Pause instantly:** click the Recall item in the status bar (bottom of the window) to pause or resume capture. The browser extension's toolbar toggle stays in sync.

**Secrets are redacted automatically:** before anything is saved or embedded for search, Recall scans for likely secrets — API keys, JWTs, `.env`-style values, credentials in URLs — and strips them. Run `Recall: Test Redaction` from the Command Palette to see it work on text you paste in yourself.
