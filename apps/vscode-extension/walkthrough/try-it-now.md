The Recall sidebar's "..." menu (top-right of the Memories panel) has three shortcuts you'll likely use most:

- **Ask Recall** — ask a question about your own past work; it answers from what it's already captured, with citations back to the source.
- **Show Daily Standup** — a draft "yesterday / today / blockers" summary generated from what you actually did.
- **Show Weekly Summary** — the same idea, rolled up over a week.

None of these need the Command Palette — they're one click away, any time.
