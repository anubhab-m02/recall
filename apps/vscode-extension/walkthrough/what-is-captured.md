Recall watches a handful of signals as you work:

- **File saves** — captured as a diff against the previous saved version, never the full file content.
- **Terminal commands** — the command line, working directory, exit code, and a truncated excerpt of the output.
- **Manual notes** — anything you explicitly save with `Recall: Save as Memory`.

Git activity, debug sessions, and diagnostics resolution are coming in a later update. Nothing here ever leaves your machine.
